# TODO list 任务清单 

> 使用 React 搭建简易TODO list APP

![License](https://img.shields.io/badge/license-MIT-green.svg)

本项目使用了create-react-app脚手架，CSS Flex进行页面布局， 并使用 useState Hook 以及 useEffect Hook实现添加 /完成 / 删除任务。
